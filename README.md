Este repositório contém os exercícios propostos durante o curso de Python da plataforma Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

Sobre o Curso:
O curso é voltado para iniciantes e cobre desde os conceitos básicos da linguagem Python até estruturas mais avançadas como tratamento de erros, manipulação de arquivos e uso de módulos.

Objetivo:
O objetivo deste repositório é consolidar o aprendizado através da prática, organizando os exercícios por módulos e disponibilizando as soluções desenvolvidas durante o curso.
